## 概要

- このフォルダは、Pugで開発をするにあたっての**HTMLのページ**に関してのDOM インターフェイスのデータで共通辞書データとなるファイルを収めるフォルダです。
- DOMインターフェイスデータファイルフォルダの下位ファルダとする事を、想定しています。
- データの意味を意識した、フォルダ構成にします。

## 使い方

上位の [データフォルダ](../../README.md) の README に記述があります。

## 内部規約

1. 主に [コーディングルール](../../../Document/20_Design/README.md) に準じます。
2. 上位の [データフォルダ](../../README.md) の README に記述があります。

## 構成

ファイルへのリンクを貼っていますが、Pugファイルにはリンク機能はありませんので、ブラウザなどの**戻る**機能を使って下さい。

- [`showpopoveroptions.json`](./showpopoveroptions.json) ShowPopoverOptions
- [`togglepopoveroptions.json`](./togglepopoveroptions.json) TogglePopoverOptions
- [`shadowrootinit.json`](./shadowrootinit.json) ShadowRootInit
