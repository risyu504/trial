# 設定ファイルフォルダ readme

この `README.md` ファイルは、このフォルダに関しての説明を記述したファイルです。

## 概要

- このフォルダは、**各処理の設定ファイル**を纏めるフォルダです。(フォルダ構成は「[フォルダ構成・方針](../Document/00_Materials/folder_policy.md)」で記しています。)
- [開発フォルダ](../README.md)の下位ファルダとする事を、想定しています。
- ツールによっては、設定ファイルの置き場所が決まっているものもあるので、ここに全ての設定ファイルを纏められる訳ではありません。
- 名称: **Config**, 区分: **Cf** です。

## 使い方

- 各ツールで対象の設定ファイルを読み込みます。読み込み方法は、ツールの使い方の方で調べて下さい。

## 内部規約

1. 主に [コーディングルール](../Document/20_Design/codingrules.md) に準じます。
2. ツールによっては、設定ファイルのファイル名が決まっている物もあるので、その場合は内部規約の例外とします。
3. このフォルダのフォルダ区分は、'**Cf**' です。

## 構成

- [`risyuconfig.json`](./risyuconfig.json) Sass, Pug 生成用制御ファイル
- `README.md` 説明ファイル(このファイルです)
