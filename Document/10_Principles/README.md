# 基本原則

この `README.md` ファイルは、このフォルダについての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の基本原則を分類して管理を容易にする事を目的としています。
- [書類倉庫](../README.md)の下位フォルダとする事を、想定しています。(書類倉庫の整理用)
- 基本原則は、設計判断の最上位規範です。「設計の設計」に相当します。
- プロジェクト文書の個別の配置は、**前提資料**の[文書体系・一覧](../00_Materials/document_structure.md)を参照。
- この `README.md` ファイルは、書類の管理も兼ねているので、基本原則の文書の新規作成, 移動, 削除の時は、必ず[一覧](#一覧)を更新して下さい。

## 一覧
