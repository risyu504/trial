# 補助資料 readme

この `README.md` ファイルは、このフォルダについての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の補助資料を分類して管理を容易にする事を目的としています。
- [書類倉庫](../README.md)の下位フォルダとする事を、想定しています。(書類倉庫の整理用)
- 補助資料は、チェックリストや調査メモ、清書前の技術メモなどの資料です。
- この `README.md` ファイルは、書類の管理も兼ねているので、補助資料の新規作成, 移動, 削除の時は、必ず[一覧](#一覧)を更新して下さい。
