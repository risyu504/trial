# コータンのおもちゃ箱 ページ管理用 データフォルダ readme

この `README.md` ファイルは、このフォルダに関しての説明を記述したファイルです。

## 概要

- このフォルダは、**コータンのおもちゃ箱 ページ管理用**の関連データファイルを収めるフォルダです。
- Webサイトを記述をする為の関連データファイルです。
- [データフォルダ](../README.md)の下位フォルダとする事を、想定しています。
- 以前は、「[ソース](../Src/README.md)」フォルダにファイルを置いていましたが、改めてフォルダを用意する事にしました。ページ情報そのものは、これまで通りページのエントリファイルと一緒に置きますが、そこからの加工データは、データフォルダ配下に置く事にしました。
- データの内容を意識しての、それに対応する構成です。

## 使い方

上位の [データフォルダ](../README.md) のreadme に記述があります。

## 内部規約

1. 主に [コーディングルール](../../Document/codingrules.md) に準じます。
2. 上位の [データフォルダ](../README.md) のreadme に記述があります。
3. 現時点では、内部作成データのみなので、規約の例外はありません。

## 構成

自由な構成の為、ここでは記しません。しいて言えば、JSONファイルを分かり易くフォルダ分けしてある位です。

- ※ データを分かり易くフォルダ分け
- `foldermap.json` 用意したデータファイルをスクリプトで纏めたデータファイル
- `sitemap.json` 用意したデータファイルを元にスクリプトで作成したデータファイル
- `README.md` 説明ファイル(このファイルです)
