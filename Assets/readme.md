# 資産フォルダ readme

この `README.md` ファイルは、このフォルダに関しての説明を記述したファイルです。

## 概要

- このフォルダは、**構成する様々な「資産」**を纏めるフォルダです。(フォルダ構成は「[フォルダ構成・方針](../Document/00_Materials/folder_policy.md)」で記しています。)
- [開発フォルダ](../README.md)の下位フォルダとする事を、想定しています。
- 名称: **Assets**, 区分: **As** です。

## 使い方

- 意味別にフォルダを用意します。
- 各ツールで対象データを読み込みます。読み込み方法は、ツールの使い方の方で調べて下さい。

## 内部規約

1. 主に [コーディングルール](../Document/20_Design/codingrules.md) に準じます。
2. 外部データを考慮して、ある程度の例外を認めます。ただ出来るならば上記のコーディングルールに合わせるように努力する事を求めます。尚、内部で作成したデータは規約に準じる必要があります。
3. このフォルダのフォルダ区分は、'**As**'(Assets) です。
4. このフォルダのファイルに関する規約は、以下の通りとする。
    1. このフォルダのファイル区分は、以下の通りとする。
        - 現在はありません。
    2. ファイル名は、外部データを考慮して自由とする。
    3. ファイルは必要に応じて、フォルダで分ける事とする。
