# 内部仕様 readme

この `README.md` ファイルは、このフォルダについての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の内部仕様を分類して管理を容易にする事を目的としています。
- [書類倉庫](../README.md)の下位フォルダとする事を、想定しています。(書類倉庫の整理用)
- 内部仕様は、コード・変数のローカル仕様、BEMや制御変数など内部向け技術文書です。
- この `README.md` ファイルは、書類の管理も兼ねているので、内部仕様の新規作成, 移動, 削除の時は、必ず[一覧](#一覧)を更新して下さい。

## 一覧

- [BEM区分 説明書](./bemclass.md) - FLOCSSを基調とした構成を考える為に、BEM区分というものを用意しててます。そのメモ書きです。
- [制御変数 説明書](./config.md) - 制御変数に関する仕様を記したものです。
