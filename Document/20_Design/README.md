# 設計方針

この `README.md` ファイルは、このフォルダについての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の設計方針を分類して管理を容易にする事を目的としています。
- [書類倉庫](../README.md)の下位フォルダとする事を、想定しています。(書類倉庫の整理用)
- 設計方針は、命名・構造・記述に関する各種設計指針です。機能や記法のルールも含みます。
- プロジェクト文書の個別の配置は、**前提資料**の[文書体系・一覧](../00_Materials/document_structure.md)を参照。
- この `README.md` ファイルは、書類の管理も兼ねているので、設計方針の文書の新規作成, 移動, 削除の時は、必ず[一覧](#一覧)を更新して下さい。
