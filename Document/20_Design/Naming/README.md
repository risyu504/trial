# 命名規則

この `README.md` ファイルは、このフォルダについての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の命名規則を分類して管理を容易にする事を目的としています。
- [設計方針](../README.md)の下位フォルダとする事を、想定しています。(設計方針の整理用)
- 書類の管理は上位フォルダで行っているので、設計方針の文書の新規作成, 移動, 削除の時は、必ず設計方針の`README.md`を更新して下さい。

## 一覧

- [命名指針](./namingpolicy.md) - 一般的な共通命名ルール。
- [フォルダ・ファイル命名規則](./namingfolder.md) - 命名形式、接尾辞、通番などの明確な規定。
- [メンバの命名規則](./namingmember.md) - Sass/JS等のメンバ名に関する命名ルール
- [セレクタの命名規則](./namingselector.md) - FLOCSS・BEMベースのセレクタ構文ルール。
