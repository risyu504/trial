# 開発環境関連資料 readme

## 概要

- このフォルダは、プロジェクト文書の開発環境関連資料を分類して管理を容易にする事を目的としています。
- [書類倉庫](../README.md)の下位フォルダとする事を、想定しています。(書類倉庫の整理用)
- 開発環境関連資料は、環境構築・実行基盤の手順やメモ、OS, Node.js, Gulp等の記録です。
- この `README.md` ファイルは、書類の管理も兼ねているので、開発環境関連資料の新規作成, 移動, 削除の時は、必ず[一覧](#一覧)を更新して下さい。
