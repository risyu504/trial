# 構造

この `README.md` ファイルは、このフォルダに関しての説明を記述したファイルです。

## 概要

- このフォルダは、プロジェクト文書の構造関連を分類して管理を容易にする事を目的としています。
- [設計方針](../README.md)の下位フォルダとする事を、想定しています。(設計方針の整理用)
- 書類の管理は上位フォルダで行っているので、設計方針の文書の新規作成, 移動, 削除の時は、必ず設計方針の`README.md`を更新して下さい。
